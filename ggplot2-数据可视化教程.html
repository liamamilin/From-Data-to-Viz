Chapter 3 ggplot2 数据可视化教程

ggplot2 是一个基于 R 语言的数据可视化包，它基于图形语法 (grammar of graphics)，允许用户以非常灵活的方式创建复杂的图形。本教程将从基础到高级的使用进行介绍，包括如何使用 ggplot2 绘制出版级别的多统计图形，并详细介绍如何调整图形细节。

3.1 基础使用

3.1.1 安装和加载

首先，确保你已经安装了 ggplot2 包。如果没有安装，可以通过以下命令安装：

# install.packages("ggplot2")

安装完成后，通过以下命令加载 ggplot2 包：

library(ggplot2)

3.1.2 创建一个简单的图形

ggplot2 的基本概念是将数据集、几何对象 (geom) 和美学属性 (aes) 结合起来创建图形。以下是一个简单的例子，展示了如何使用 ggplot2 绘制散点图：

# 加载ggplot2包
library(ggplot2)

# 使用mtcars数据集
data(mtcars)

# 创建一个散点图，展示wt和mpg之间的关系
ggplot(mtcars, aes(x = wt, y = mpg)) + 
  geom_point()

3.1.3 添加图层

ggplot2 的一个强大功能是可以通过添加更多的图层来丰富图形，比如添加线条、平滑线、文本等。

# 继续上面的例子，添加一条回归线
ggplot(mtcars, aes(x = wt, y = mpg)) + 
  geom_point() +
  geom_smooth(method = "lm")
## `geom_smooth()` using formula = 'y ~ x'

3.2 高级使用

3.2.1 自定义主题

ggplot2 允许用户通过修改主题来自定义图形的外观，例如更改文本的字体、大小、颜色等。

# 使用theme函数自定义图形
ggplot(mtcars, aes(x = wt, y = mpg)) + 
  geom_point() +
  geom_smooth(method = "lm") +
  theme_minimal() +
  theme(text = element_text(family = "Helvetica", size = 12, color = "black"))
## `geom_smooth()` using formula = 'y ~ x'

3.2.2 多统计图形

ggplot2 可以创建包含多个统计信息的复杂图形，例如，同时展示数据的分布、中位数、平均值等。

# 使用mtcars数据集，展示每个汽缸数(cyl)的mpg分布
ggplot(mtcars, aes(x = factor(cyl), y = mpg)) + 
  geom_boxplot() +
  stat_summary(fun = mean, geom = "point", shape = 20, size = 3, color = "red") +
  labs(title = "汽缸数对汽车燃油效率的影响", x = "汽缸数", y = "燃油效率 (mpg)")

3.2.3 调整图形细节

ggplot2 提供了丰富的选项来调整图形的细节，如调整坐标轴、图例、标签等。

# 调整坐标轴和图例
ggplot(mtcars, aes(x = wt, y = mpg, color = factor(cyl))) + 
  geom_point(size = 3) +
  scale_color_manual(values = c("red", "blue", "green")) +
  labs(title = "重量对燃油效率的影响", x = "重量", y = "燃油效率 (mpg)", color = "汽缸数") +
  theme_light() +
  theme(axis.title.x = element_text(size = 14, face = "bold"),
        axis.title.y = element_text(size = 14, face = "bold"),
        legend.title = element_text(size = 12))
